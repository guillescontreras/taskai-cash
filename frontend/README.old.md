# Frontend PWA

Progressive Web App para TaskAI Cash.

## Tecnologías

- React/Vue.js
- PWA capabilities
- Responsive design
- AdMob integration

## Features

- Lista de tareas
- Generación de tareas con IA
- Sistema de recompensas
- Integración de ads
- Perfil de usuario

## Desarrollo

```bash
npm install
npm run dev
```
