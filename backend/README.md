# Backend

Lambda functions para TaskAI Cash.

## Functions

- `auth/` - Autenticación y autorización
- `tasks/` - CRUD de tareas
- `users/` - Gestión de usuarios
- `payments/` - Integración con Stripe
- `ai/` - Integración con Amazon Q
- `ads/` - Gestión de publicidad

## Estructura

```
backend/
├── auth/
├── tasks/
├── users/
├── payments/
├── ai/
└── ads/
```
